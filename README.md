# springboot expense tracker application

## How to run the application

```shell
git clone https://github.com/manjosh1990/expensemanager.git
cd xpensive-app
./run.sh start
./run.sh stop

./run.sh start_infra
./run.sh stop_infra
```

* To start only dependent services
```shell
$ ./run.sh start_infra
$ ./run.sh stop_infra
```
